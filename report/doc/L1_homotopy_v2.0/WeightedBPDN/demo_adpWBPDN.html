DESCRIPTION 

 demo_adpWBPDN.m

 Solves the following basis pursuit denoising (BPDN) problem
 min_x  \Sum \w_i |x_i| + 1/2*||y-Ax||_2^2

 while adaptively selecting the weights w_i
