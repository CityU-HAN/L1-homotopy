DESCRIPTION 

 demo_dynamicX

 Solves the following BPDN problem
 min_x  tau\|x\|_1 + 1/2*||y-Ax||_2^2
 
 and updates the solution using updated measurements 
 for a slighly modified x

 Email: sasif@gatech.edu
