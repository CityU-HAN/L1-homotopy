DESCRIPTION 

 cdfspline.m

 Filter coefficients for Cohen, Daubechies, Faveau biorth spline filters
 Usage : [h0,g0] = cdfspline(N,M)
 h0 - bspline poly of order N
 g0 - poly using rest of factors from daubpoly((N+M)/2)
 h0 and g0 are scaling filters
